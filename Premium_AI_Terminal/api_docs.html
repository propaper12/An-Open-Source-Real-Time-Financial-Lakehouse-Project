Dinamik Veri Maskeleme (DDM)

Sistemimiz, yüzlerce farklı URL'ye istek atmanızı engelleyen tek bir kurumsal ağ geçidine (API Gateway) sahiptir. İsteği attığınız anda gönderdiğiniz X-API-Key arka planda çözümlenir. Paketinizin seviyesine (Free, Pro, VIP) göre veri otomatik olarak filtrelenir (maskelenir) veya canlı/gecikmeli olarak sunulur.

Kimlik Doğrulama & Hız Limitleri

API anahtarınızı isteklerinize nasıl ekleyeceğiniz ve Redis tabanlı Tier-Based Rate Limiting sistemimizin nasıl çalıştığı aşağıda açıklanmıştır. Kurallara uyulmaması durumunda hesaplarınız kalıcı olarak engellenebilir.

REST (HTTP) İSTEKLERİ

API anahtarı header içerisinde gönderilmelidir:

X-API-Key: sk_live_YOUR_KEY

WEBSOCKET İSTEKLERİ

API anahtarı URL parametresi olarak eklenmelidir:

?api_key=sk_live_YOUR_KEY

JSON Veri Şeması ve Maskeleme Kuralları

Aşağıdaki tabloda, uç noktamızın döndürdüğü JSON verisinin yapısı bulunmaktadır. Eğer PRO kullanıcısıysanız, VIP verilere (AI, CVD) erişiminiz engellenir ve bu sütunlar arka planda record.pop() edilerek JSON'dan silinir.

REST API (Free & PRO İçin)

Piyasa verisini statik olarak "çekmek" (polling) için kullanılır. Gönderdiğiniz API anahtarına göre arka plandaki `ingestion_api.py` sistemi şunları yapar:
1. FREE Anahtarsa: Veritabanında 15 dakika geriye gider, veriyi çeker ve verir.
2. PRO Anahtarsa: Canlı veriyi çeker ancak AI ve CVD verilerini JSON'dan silerek verir.

GET https://api.radarglobal.com/api/v1/market/{symbol}
fetch_rest_data.pyPython
import requests

# API anahtarınız neyse (FREE, PRO, VIP), sistem ona göre davranır.
API_KEY = "sk_live_YOUR_KEY"
SYMBOL = "BTCUSDT"
URL = f"https://api.radarglobal.com/api/v1/market/{SYMBOL}"

headers = {"X-API-Key": API_KEY}
response = requests.get(URL, headers=headers)

if response.status_code == 200:
    payload = response.json()
    print(f"Kullanıcı Seviyesi:", payload['tier'])     # Çıktı: FREE veya PRO
    print(f"Sistem Gecikmesi:", payload['delay'])    # Çıktı: '15 Minutes' veya 'Real-Time'
    
    data = payload['data']
    print(f"Fiyat:", data['average_price'])
    print(f"Zaman:", data['processed_time'])
    
elif response.status_code == 429:
    print("🚨 Redis Limit: Çok hızlı istek atıyorsunuz!")
elif response.status_code == 403:
    print("⛔ BAN: Günlük limiti aştınız.")

🤖 Kurumsal Seviye: Otonom AI Botu Kodlama

Sıradan bir bot sadece fiyat düşüşüne bakar. RadarPro VIP kullanıcısı ise Akıllı Para Akışını (CVD) ve Yapay Zeka Hedefini (predicted_price) kullanarak aşağıdaki gibi profesyonel bir asenkron bot yazar.

radar_quant_bot.pyPython (Asyncio)
import asyncio
import websockets
import json
# import ccxt  # Binance veya Bybit emir gönderimi için kütüphane

class RadarQuantBot:
    def __init__(self, api_key, symbol):
        self.api_key = api_key
        self.symbol = symbol
        self.in_position = False
        self.entry_price = 0
        # self.exchange = ccxt.binance({'apiKey': 'BorsaKey', 'secret': 'BorsaSecret'})

    async def listen_and_trade(self):
        url = f"wss://api.radarglobal.com/api/v1/stream/{self.symbol}?api_key={self.api_key}"
        
        try:
            async with websockets.connect(url) as ws:
                print(f"🟢 Kafka HFT Bağlantısı Kuruldu: {self.symbol} dinleniyor...")
                
                while True:
                    response = json.loads(await ws.recv())
                    
                    # VIP Anahtar kontrolü başarısız olursa API Gateway bağlantıyı kapatır
                    if 'error' in response:
                        print(f"🔴 Güvenlik İhlali: {response['error']}")
                        break

                    data = response['data']
                    price = float(data['average_price'])
                    ai_target = float(data.get('predicted_price', price))
                    cvd = float(data.get('cvd', 0))
                    
                    # ========================================================
                    # 🧠 OTONOM KARAR MOTORU (ALGORITHM)
                    # ========================================================
                    
                    # LONG (ALIM) STRATEJİSİ:
                    # Şart 1: XGBoost AI Modeli, fiyatın %2 artacağını tahmin etmeli
                    # Şart 2: CVD > 100,000 olmalı (Piyasa Yapıcılar mal topluyor olmalı)
                    
                    if not self.in_position:
                        if ai_target > (price * 1.02) and cvd > 100000:
                            print(f"\n🚀 AI ALIM SİNYALİ ONAYLANDI!")
                            print(f"Fiyat: {price} | AI Hedef: {ai_target} | Net Akış (CVD): {cvd}")
                            
                            # self.exchange.create_market_buy_order(self.symbol, amount=0.1)
                            self.in_position = True
                            self.entry_price = price
                            
                    # ÇIKIŞ (TAKE PROFIT / STOP LOSS) STRATEJİSİ:
                    elif self.in_position:
                        # Kar Al (%1.5 yukarıda)
                        if price >= self.entry_price * 1.015:
                            print(f"\n💰 TAKE PROFIT: Fiyat {price} seviyesine ulaştı. Satış yapılıyor.")
                            self.in_position = False
                            
                        # Zarar Kes (AI Yanılırsa %0.5 aşağıda kol kes)
                        elif price <= self.entry_price * 0.995:
                            print(f"\n🩸 STOP LOSS PATLADI: Fiyat {price}. Çıkış yapılıyor.")
                            self.in_position = False
                            
        except Exception as e:
            print(f"Bağlantı koptu veya hata oluştu: {e}")

if __name__ == "__main__":
    # Kullanıcı portalından aldığınız VIP anahtarı yapıştırın
    bot = RadarQuantBot(api_key="sk_live_VIP_KEY_HERE", symbol="BTCUSDT")
    asyncio.run(bot.listen_and_trade())
