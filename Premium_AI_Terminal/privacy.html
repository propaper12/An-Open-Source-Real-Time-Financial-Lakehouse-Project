Gizlilik Politikası

Son Güncelleme: 6 Mart 2026

1. Veri Toplama ve Kullanımı

Radar Global olarak, sunduğumuz API ve Terminal hizmetlerini sağlarken minimum düzeyde kişisel veri toplama prensibini benimsiyoruz. Sisteme kayıt olduğunuzda sadece kullanıcı adı, e-posta adresi ve güvenli kriptografik algoritmalarla (SHA-256) şifrelenmiş parolanız saklanır. API İstekleriniz sistem performansını ölçmek amacıyla geçici log dosyalarında tutulur.

2. API Anahtarı Güvenliği

Oluşturduğunuz sk_live_ önekli API anahtarlarınızın güvenliği tamamen sizin sorumluluğunuzdadır. RadarPro altyapısı bu anahtarları üçüncü şahıslarla paylaşmaz. Bir sızıntı tespit edildiğinde hesap portalı üzerinden eski anahtarınızı iptal edip yenisini oluşturabilirsiniz.

3. Çerezler (Cookies) ve LocalStorage

Sistemimiz, oturumunuzu açık tutmak ve API anahtarınızı terminal üzerinde otomatik tanımak için tarayıcınızın LocalStorage (Yerel Depolama) özelliğini kullanır. Üçüncü taraf reklam takip çerezleri kullanılmamaktadır.

4. Veri Paylaşımı

Verileriniz, yasal bir zorunluluk (mahkeme kararı vb.) olmadığı sürece hiçbir kurum, kuruluş veya reklam ajansına satılmaz, kiralanamaz veya devredilemez.
